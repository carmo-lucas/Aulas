3.2 Operadores Lógicos

Operador Descrição
> Maior que
>= Maior ou igual a
< Menor que
<= Menor ou igual a
== Exatamente igual a
!= Não igual a
!x Negação
x | y x OU y
x & y x E y
isTRUE(x) Testa se x é TRUE

Estes operadores irão sempre retornar valores lógicos do tipo TRUE ou FALSE.

3.2.1 Exemplos
